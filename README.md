# Framework 
Contiene ejemplos de cómo utilizar frameworks (específicamente Bootstrap) para el desarrollo web.
Aprenderemos a utilizar componentes como Carruseles, Modales y Cards.

> Gracias a estas herramientas podemos crear interfaces modernas y responsivas
> de una manera más rápida y sencilla.

## Ejemplo de código (Bootstrap)
```html
<div id="carouselExample" class="carousel slide">
    <div class="carousel-inner">
        <div class="carousel-item active">
            <img src="..." class="d-block w-100" alt="...">
        </div>
        <div class="carousel-item">
            <img src="..." class="d-block w-100" alt="...">
        </div>
    </div>
    <button class="carousel-control-prev" type="button" data-bs-target="#carouselExample" data-bs-slide="prev">
        <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    </button>
</div>
```
> En este bloque de código se muestra la estructura básica de un Carrusel de Bootstrap, 
> el cual permite mostrar múltiples imágenes en un mismo espacio de forma dinámica.
